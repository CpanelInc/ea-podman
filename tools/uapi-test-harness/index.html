ea-podman UAPI test harness

Manual browser QA tool for the Cpanel::API::EAPodman UAPI module (CPANEL-54037). Calls go through the bundled local proxy (proxy.py) because cpsrvd does not send Access-Control-* headers, so a browser cannot call it cross-origin directly. Run python3 proxy.py in this directory first, then fill in the config below.
